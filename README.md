## 🔧 Tehnologije
- C#
- React
- Docker Compose
- Redis
- MongoDB SUBP
- SQLServer SUBP

- Microsoft ASP.NET Identity
- JWT

## ⚙️ Instalacije

##  🔨 Pokretanje
```bash
docker-compose -f docker-compose.yml -f docker-compose.override.yml up -d --build
npm install
npm start
```
